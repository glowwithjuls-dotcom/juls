# AGENTS.md

## Project
Titi’s Cosmetics ecommerce website.

## Goal
Build a clean MVP first for a nationwide Ghana cosmetics store with Mobile Money payments.

## Priorities
1. Functional storefront
2. Reliable checkout
3. Admin usability
4. Mobile-first design
5. Clean code and simple deployment

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- Cloudinary
- Paystack
- Vercel

## Rules
- Read BUILD_SPEC.md before coding
- Use App Router
- Use TypeScript everywhere practical
- Use Prisma for all DB models
- Keep components reusable
- Use environment variables for secrets
- Never expose secret keys client-side
- Make UI mobile-first
- Seed sample products from SEED_PRODUCTS.csv
- Build MVP only; do not implement advanced AI features yet

## Required pages
- Home
- Shop
- Product details
- Cart
- Checkout
- Contact
- Order tracking
- Admin login
- Admin dashboard

## Business constraints
- Ghana-focused checkout
- GHS currency
- Nationwide delivery
- Mobile Money first
- WhatsApp contact support

## Expected output
- Working local app
- Clean README
- Seeded sample catalog
- Deployment-ready structure
