# Titi's Cosmetics Website

This repository contains the build specification for the Titi’s Cosmetics ecommerce website.

## Tech stack
Next.js
TypeScript
Tailwind CSS
PostgreSQL
Prisma
Cloudinary
Paystack
Vercel

## Setup overview
1. Create Next.js project
2. Configure database
3. Run Prisma migrations
4. Seed products from SEED_PRODUCTS.csv
5. Configure Paystack
6. Deploy to Vercel
