# BUILD_SPEC.md

## Business summary
Titi’s Cosmetics is a wholesale and retail cosmetics shop serving customers across Ghana.
The website should allow customers to browse products, place orders, pay with Mobile Money,
and contact the shop easily.

## MVP features

### Storefront
- Homepage with hero banner and featured products
- Shop page with product grid and search
- Product details page
- Cart page
- Checkout page
- Contact page
- Order tracking page

### Admin
- Admin login
- Dashboard
- Product CRUD
- Inventory management
- Order list and status updates

### Checkout fields
- Full name
- Phone
- Email optional
- Region
- City/Town
- Address
- Payment method

### Payment methods
Primary: Paystack
- Mobile Money
- Card

Optional: Pay on delivery

### Delivery
- Nationwide delivery in Ghana
- Delivery fee calculated by region

### Order statuses
- Pending
- Confirmed
- Dispatched
- Delivered
- Cancelled

## UX requirements
- Mobile-first design
- Beauty store theme
- Clear call-to-action buttons
- Fast loading pages
